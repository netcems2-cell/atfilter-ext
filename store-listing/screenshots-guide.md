# Screenshot Guide for Store Submissions

## Requirements
- **Chrome Web Store:** 1280x800 or 640x400, PNG or JPEG, 1-5 screenshots
- **Firefox Add-ons:** Any reasonable size, PNG or JPEG, 1-5 screenshots

## Screenshots to Take (5 recommended)

### Screenshot 1: Popup — Keyword Entry
**What to show:** The @Filter popup with a few example keywords entered
**How:**
1. Open any news site (e.g., CNN, BBC, Reddit)
2. Click the @Filter icon in the toolbar
3. Enter 3-4 example keywords (e.g., "clickbait", "spoiler", "politics")
4. Take screenshot of the popup
**Resize to:** 1280x800 (add padding/background if needed)

### Screenshot 2: Before & After — Filtering in Action
**What to show:** Side-by-side or annotated view showing content being filtered
**How:**
1. Go to a news site with many headlines
2. Take screenshot WITHOUT filtering (or with filter off)
3. Enable filtering with a relevant keyword
4. Take screenshot WITH filtering active — matching items hidden
5. Combine into a single 1280x800 image with "Before" / "After" labels
**Tip:** Use a common keyword like "trump" or "politics" that has many matches on news sites

### Screenshot 3: Settings Page
**What to show:** The settings page with community participation toggle
**How:**
1. Right-click @Filter icon > Options (or click Settings link in popup)
2. Show the settings page with community participation toggle ON
3. Capture the full settings view
**Resize to:** 1280x800

### Screenshot 4: Onboarding Page
**What to show:** The clean onboarding welcome screen
**How:**
1. Open onboarding.html directly in browser
2. Capture the full onboarding page showing the community message and steps
**Resize to:** 1280x800

### Screenshot 5: @Map Website (Optional)
**What to show:** The @Map visualization on atfilter.com
**How:**
1. Go to atfilter.com
2. Scroll to the map section
3. Switch to "Top Keywords" view if live data is available
4. Capture the map with the clean interface
**Resize to:** 1280x800

## Additional Assets

### Promotional Tile (Chrome Web Store, optional)
- Small: 440x280
- Large: 920x680
- Marquee: 1400x560
**Suggestion:** Orange background with @Filter logo and tagline "Content filtering for the open web"

## Tips
- Use a clean browser profile (no other extensions visible)
- Use light theme for consistency
- Ensure no personal data is visible in screenshots
- Keep the browser window at a standard size
- Consider using a screenshot tool that can output exact dimensions
